HTML+JS демонстрация

Ниже пример интерактивного блока, в котором при клике меняется случайное число. Это простая демонстрация динамического взаимодействия с пользователем с помощью JavaScript.

Нажми меня

// Получаем ссылки на элементы страницы
const demoBox = document.getElementById('demoBox');
const resetBtn = document.getElementById('resetBtn');

// Функция для генерации случайного числа от 1 до 100
function randomNumber() {
  return Math.floor(Math.random() * 100) + 1;
}

// Обработчик клика по блоку
demoBox.addEventListener('click', () => {
  demoBox.textContent = randomNumber();  // Заменяем текст случайным числом
});

// Обработчик клика по кнопке сброса
resetBtn.addEventListener('click', () => {
  demoBox.textContent = 'Нажми меня';  // Возвращаем исходный текст
});

// Обработчик нажатия клавиш для доступности
demoBox.addEventListener('keydown', (event) => {
  if(event.key === 'Enter' || event.key === ' ') {
    event.preventDefault();              // Отменяем прокрутку при пробеле
    demoBox.textContent = randomNumber();
  }
});
Пояснения к коду:
  • getElementById — ищет элемент по ID на странице.
  • Math.random() генерирует число от 0 до 1 (не включая 1).
  • Math.floor() округляет вниз до целого числа.
  • Обработчики событий click и keydown запускают функции при клике и нажатии клавиш.
  • Атрибуты tabindex и role="button" делают блок доступным для навигации с клавиатуры.
  • Используем event.preventDefault(), чтобы отменить стандартное поведение браузера при пробеле.
